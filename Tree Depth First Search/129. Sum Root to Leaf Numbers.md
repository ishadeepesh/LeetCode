## üå≥ 129. Sum Root to Leaf Numbers

### üìù Problem Statement  
Given the `root` of a binary tree **containing digits (0‚Äì9)**, each root-to-leaf path forms a number (e.g., `1 -> 2 -> 3` becomes `123`).  
Return the **total sum** of all root-to-leaf numbers.

---

### üì• Example  

**Input:**  
`root = [1,2,3]`  

**Explanation:**  
- Path 1 ‚Üí 2 = 12  
- Path 1 ‚Üí 3 = 13  
**Sum = 12 + 13 = 25**  

**Output:**  
`25`

---

### üí° Approach  

1. Use **DFS (Depth-First Search)** to traverse each root-to-leaf path.
2. At each step, pass the **cumulative number formed** by multiplying current total by 10 and adding current node‚Äôs value.
3. When a **leaf** node is reached, return the number formed.
4. Sum all such values from left and right subtrees.

---

### ‚úÖ Python Code  
```python
class Solution:
    def sumNumbers(self, root: Optional[TreeNode]) -> int:
        return self.dfs(root, 0)
    
    def dfs(self, root: TreeNode, total: int) -> int:
        if not root:
            return 0

        total = total * 10 + root.val

        if not root.left and not root.right:  # Leaf node
            return total

        # Recur for left and right subtrees
        return self.dfs(root.left, total) + self.dfs(root.right, total)
```
‚è±Ô∏è Time Complexity
O(n) ‚Äî Every node is visited once.

üíæ Space Complexity
O(h) ‚Äî Where h is the height of the tree (due to recursion stack).
