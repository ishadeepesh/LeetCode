## üìö 112. Path Sum

### üìù Problem Statement  
Given the root of a binary tree and an integer `targetSum`, return `true` if the tree has a **root-to-leaf** path such that adding up all the values along the path equals `targetSum`.

A **leaf** is a node with no children.

---

### üì• Example  

**Input:**  
`root = [5,4,8,11,null,13,4,7,2,null,null,null,1]`, `targetSum = 22`  

**Output:**  
`true`  

**Explanation:**  
The root-to-leaf path `5 ‚Üí 4 ‚Üí 11 ‚Üí 2` sums to 22.

---

### üí° Approach  
- Traverse the tree using recursion.
- At each node, subtract the current node‚Äôs value from `targetSum`.
- If the node is a **leaf**, check if `targetSum == node.val`.
- Return `True` if any path satisfies the condition.

---

### ‚úÖ Python Code
```python
class Solution:
    def hasPathSum(self, root: Optional[TreeNode], targetSum: int) -> bool:
        if not root:
            return False
        
        # Check if current node is a leaf
        if not root.left and not root.right:
            return root.val == targetSum
        
        # Recurse for left and right with updated target
        return (self.hasPathSum(root.left, targetSum - root.val) or
                self.hasPathSum(root.right, targetSum - root.val))
```
‚è±Ô∏è Time Complexity
O(n) ‚Äî Visit every node once in the worst case.

üíæ Space Complexity
O(h) ‚Äî h is the height of the tree (recursive call stack).
