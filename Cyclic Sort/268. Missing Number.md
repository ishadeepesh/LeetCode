## ‚úÖ 268. Missing Number

---

### üßæ Question:
Given an array `nums` containing `n` **distinct** numbers in the range `[0, n]`, return the **only** number in the range that is **missing** from the array.

---

### üîπ Example:

**Input:**
nums = [3, 0, 1]

**Output:**
2

**Explanation:**
- The input has 3 elements, so the full range is `[0, 1, 2, 3]`.
- The number `2` is missing from the array.

### ‚úÖ Code (Cyclic Sort Approach):

```python
class Solution:
    def missingNumber(self, nums: List[int]) -> int:
        i = 0
        n = len(nums)
        while i < n:
            correctIndex = nums[i]
            if correctIndex < n and nums[i] != nums[correctIndex]:
                nums[i], nums[correctIndex] = nums[correctIndex], nums[i]
            else:
                i += 1
        for i in range(n):
            if nums[i] != i:
                return i
        return n
```
---
### ‚úÖ Alternate Code (XOR Approach):
```python
class Solution:
    def missingNumber(self, nums: List[int]) -> int:
        ans = 0
        n = len(nums)
        for i in nums:
            ans ^= i
        for i in range(1, n + 1):
            ans ^= i
        return ans
```
‚è±Ô∏è Time Complexity:
- Cyclic Sort: O(n)

- XOR: O(n)

üì¶ Space Complexity:
- Both approaches use O(1) space (in-place or constant extra variables).
