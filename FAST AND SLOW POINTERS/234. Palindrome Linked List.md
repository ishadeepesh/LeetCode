## ğŸ’¡ Approach: Fast and Slow Pointer + Reverse Second Half

1. Use two pointers (`slow` and `fast`) to find the middle of the list.
2. Reverse the second half of the list starting from the `slow` pointer.
3. Compare the first half and reversed second half node by node.
4. If all corresponding nodes match, return `True`. Otherwise, return `False`.

## âœ… Python Code

```python
class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        slow = head
        fast = head

        # Step 1: Find the middle using slow and fast pointers
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next

        # Step 2: Reverse the second half of the list
        prev = None
        while slow:
            temp = slow.next
            slow.next = prev
            prev = slow
            slow = temp

        # Step 3: Compare first half and reversed second half
        left, right = head, prev
        while right:
            if left.val != right.val:
                return False
            left = left.next
            right = right.next

        return True
```
