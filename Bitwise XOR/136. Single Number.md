Given a non-empty array of integers `nums`, **every element appears twice except for one**.  
Find that **single one**.

- Linear runtime complexity (**O(n)**), and

### âœ… Code:

```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        ans = 0
        for i in nums:
            ans = ans ^ i
        return ans
```
