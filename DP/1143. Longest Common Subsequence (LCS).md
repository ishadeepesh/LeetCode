# 🔗 1143. Longest Common Subsequence (LCS)

## 🧩 Problem Statement

Given two strings `text1` and `text2`, return the length of their **Longest Common Subsequence (LCS)**.

A subsequence is formed by deleting characters **without changing order**.

---

## 📌 Example

### Input:
text1 = "abcde"  
text2 = "ace"  

### Output:
3  

### 💡 Explanation:
LCS = "ace"

---

## 🧠 Key Idea

For every pair of indices `(i, j)`:

- If characters match → include it  
- Else → skip one character and try both options  

---

# 🚀 Approach 1: Recursion (Brute Force)

## 💡 Idea

Try all possibilities:
- If match → move diagonally  
- Else → explore both directions  

---

## 🧾 Code

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        def LCS(m, n):
            if m == 0 or n == 0:
                return 0

            if text1[m-1] == text2[n-1]:
                return 1 + LCS(m-1, n-1)
            else:
                return max(LCS(m-1, n), LCS(m, n-1))

        return LCS(len(text1), len(text2))
```

---

## ⏱️ Complexity

- Time: `O(2^(m+n))` ❌ (very slow)  
- Space: recursion stack  

---

# ⚡ Approach 2: Memoization (Top-Down DP)

## 💡 Idea

Same recursion, but store results to avoid recomputation.

---

## 🧾 Code

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        t = [[-1]*(n+1) for _ in range(m+1)]

        def LCS(m, n):
            if m == 0 or n == 0:
                return 0

            if t[m][n] != -1:
                return t[m][n]

            if text1[m-1] == text2[n-1]:
                t[m][n] = 1 + LCS(m-1, n-1)
            else:
                t[m][n] = max(
                    LCS(m-1, n),
                    LCS(m, n-1)
                )

            return t[m][n]

        return LCS(m, n)
```

---

## ⏱️ Complexity

- Time: `O(m * n)` ✅  
- Space: `O(m * n)`  

---

# 🧱 Approach 3: Bottom-Up (Tabulation)

## 💡 Idea

Build a DP table iteratively:

- `t[i][j]` = LCS length for first `i` chars of text1 and first `j` chars of text2  

---

## 🧾 Code

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)

        t = [[0]*(n+1) for _ in range(m+1)]

        for i in range(1, m+1):
            for j in range(1, n+1):
                if text1[i-1] == text2[j-1]:
                    t[i][j] = 1 + t[i-1][j-1]
                else:
                    t[i][j] = max(t[i-1][j], t[i][j-1])

        return t[m][n]
```

---

## ⏱️ Complexity

- Time: `O(m * n)`  
- Space: `O(m * n)`  

---

# 🔥 Summary

| Approach | Type | Time |
|--------|------|------|
| Recursion | Brute force | ❌ Exponential |
| Memoization | Top-down DP | ✅ O(m*n) |
| Tabulation | Bottom-up DP | ✅ O(m*n) |

---

## 💬 Key Insight

- Match → diagonal ↖  
- No match → max(top ↑, left ←)

👉 That’s the entire LCS logic.

---

## ⚠️ Common Mistakes

- Mixing recursion with loops ❌  
- Returning inside loops ❌  
- Wrong DP initialization ❌
