# 🔗 1092. Shortest Common Supersequence (SCS)

## 🧩 Problem Statement

Given two strings `str1` and `str2`, return the **shortest string** that has both `str1` and `str2` as subsequences.

👉 A supersequence contains both strings (not necessarily contiguously).  
👉 We want the **minimum length** possible.

---

## 📌 Example

### Input:
str1 = "abac"  
str2 = "cab"  

### Output:
"cabac"  

---

## 🧠 Key Idea

👉 SCS is closely related to **LCS (Longest Common Subsequence)**

Formula:
```
Length of SCS = len(str1) + len(str2) - LCS(str1, str2)
```

---

# 🚀 Approach: LCS + Backtracking

## 💡 Idea

1. Build LCS DP table  
2. Backtrack:
   - If characters match → include once  
   - Else → include the character from the direction we move  

---

## 🧾 Code (Python)

```python
class Solution:
    def shortestCommonSupersequence(self, str1: str, str2: str) -> str:
        m, n = len(str1), len(str2)

        # Step 1: Build LCS table
        t = [[0]*(n+1) for _ in range(m+1)]

        for i in range(1, m+1):
            for j in range(1, n+1):
                if str1[i-1] == str2[j-1]:
                    t[i][j] = 1 + t[i-1][j-1]
                else:
                    t[i][j] = max(t[i-1][j], t[i][j-1])

        # Step 2: Backtrack to build SCS
        i, j = m, n
        result = []

        while i > 0 and j > 0:
            if str1[i-1] == str2[j-1]:
                result.append(str1[i-1])
                i -= 1
                j -= 1
            else:
                if t[i-1][j] > t[i][j-1]:
                    result.append(str1[i-1])
                    i -= 1
                else:
                    result.append(str2[j-1])
                    j -= 1

        # Add remaining characters
        while i > 0:
            result.append(str1[i-1])
            i -= 1

        while j > 0:
            result.append(str2[j-1])
            j -= 1

        return "".join(result[::-1])
```

---

## 🔍 Intuition

While backtracking:

- ✅ Match → include once  
- ❌ No match → include whichever side we move  

👉 Ensures:
- No duplicates  
- Minimum length  

---

## 🎯 Example Walkthrough

str1 = "abac"  
str2 = "cab"  

LCS = "ab"  

Backtracking builds:
```
c a b a c
```

Final answer:
```
"cabac"
```

---

## ⏱️ Complexity

- Time: `O(m * n)`  
- Space: `O(m * n)`  

---

## 🔥 Key Difference from LCS

| Problem | Goal |
|--------|------|
| LCS | Common subsequence |
| SCS | Merge both strings with minimum length |

---

## 💬 Pro Tip

👉 Think SCS as:
> “Merge both strings while preserving order and avoiding duplication”

---

## ⚠️ Common Mistakes

- Forgetting to append remaining characters ❌  
- Confusing with LCS string ❌  
- Not reversing final result ❌  

---

## 🧠 Final Insight

👉 LCS helps avoid duplication  
👉 Rest is just merging intelligently  

That’s the whole trick.
