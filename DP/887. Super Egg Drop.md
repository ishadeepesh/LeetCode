# 🥚 Super Egg Drop (LeetCode 887)

## 🧩 Problem Statement

You are given:

* `k` eggs
* `n` floors

👉 Find the **minimum number of moves** needed to determine the highest floor from which an egg can be dropped **without breaking**.

---

## 🧠 Key Idea

At any floor `x`:

* Egg **breaks** → check below → `(k-1, x-1)`
* Egg **doesn't break** → check above → `(k, n-x)`

We take:

```
1 + max(break_case, no_break_case)
```

👉 Because we care about **worst case**

---

# 🚀 Approach 1: Pure Recursion (Brute Force)

## 💡 Idea

Try dropping from every floor

---

## 🧾 Code

```python
class Solution:
    def superEggDrop(self, k: int, n: int) -> int:
        if n == 0 or n == 1:
            return n
        if k == 1:
            return n

        mn = float('inf')

        for i in range(1, n + 1):
            temp = 1 + max(
                self.superEggDrop(k - 1, i - 1),
                self.superEggDrop(k, n - i)
            )
            mn = min(mn, temp)

        return mn
```

---

## ⏱️ Complexity

* Time: Exponential ❌
* Space: recursion stack

👉 **TLE**

---

# ⚡ Approach 2: Memoization (Top-Down DP)

## 💡 Idea

Store results for `(k, n)` to avoid recomputation

---

## 🧾 Code

```python
class Solution:
    def superEggDrop(self, k: int, n: int) -> int:
        self.t = [[-1] * (n + 1) for _ in range(k + 1)]
        return self.solve(k, n)

    def solve(self, k, n):
        if n == 0 or n == 1:
            return n
        if k == 1:
            return n

        if self.t[k][n] != -1:
            return self.t[k][n]

        mn = float('inf')

        for i in range(1, n + 1):
            # break case
            if self.t[k-1][i-1] != -1:
                temp1 = self.t[k-1][i-1]
            else:
                temp1 = self.solve(k-1, i-1)
                self.t[k-1][i-1] = temp1

            # no break case
            if self.t[k][n-i] != -1:
                temp2 = self.t[k][n-i]
            else:
                temp2 = self.solve(k, n-i)
                self.t[k][n-i] = temp2

            temp = 1 + max(temp1, temp2)
            mn = min(mn, temp)

        self.t[k][n] = mn
        return mn
```

---

## ⏱️ Complexity

* Time: ~O(n³) ⚠️
* Space: O(k * n)

👉 May still TLE for large inputs

---

# 🚀 Approach 3: Binary Search Optimization (Optimal)

## 💡 Idea

Instead of checking all floors:

* `break_case` increases with floor
* `no_break` decreases

👉 Use **binary search** to find optimal point

---

## 🧾 Code

```python
class Solution:
    def superEggDrop(self, k: int, n: int) -> int:
        self.dp = {}
        return self.solve(k, n)

    def solve(self, k, n):
        if n == 0 or n == 1:
            return n
        if k == 1:
            return n

        if (k, n) in self.dp:
            return self.dp[(k, n)]

        low, high = 1, n
        ans = float('inf')

        while low <= high:
            mid = (low + high) // 2

            break_case = self.solve(k - 1, mid - 1)
            no_break = self.solve(k, n - mid)

            temp = 1 + max(break_case, no_break)

            if break_case > no_break:
                high = mid - 1
            else:
                low = mid + 1

            ans = min(ans, temp)

        self.dp[(k, n)] = ans
        return ans
```

---

## ⏱️ Complexity

* Time: **O(k * n * log n)** ✅
* Space: O(k * n)

👉 **Best interview answer**

---

# 🔥 Summary

| Approach      | Time          | Space  | Status     |
| ------------- | ------------- | ------ | ---------- |
| Recursion     | Exponential ❌ | Stack  | TLE        |
| Memoization   | ~O(n³) ⚠️     | O(k*n) | Acceptable |
| Binary Search | O(n log n) ✅  | O(k*n) | Optimal    |

---

# 🧠 Key Insight

👉 This is **MCM-style partition DP**

But optimized using:

```
Monotonic behavior → Binary Search
```

# 💬 Interview Tip

Say this:
