# 🔀 87. Scramble String

## 🧩 Problem Statement

We can scramble a string `s` using:

1. Split string into two non-empty parts  
2. Optionally swap them  
3. Recursively apply the same process  

👉 Given `s1` and `s2`, return **True** if `s2` is a scrambled string of `s1`

---

## 📌 Example

### Input:
s1 = "great"  
s2 = "rgeat"  

### Output:
True  

---

## 🧠 Key Idea

👉 This is a **partition + recursion problem (similar to MCM)**

At every split:
```
Try:
1. No swap
2. Swap
```

---

# 🚀 Approach 1: Recursion (Brute Force)

## 💡 Idea

For every split index `i`:
- Check **no swap case**
- Check **swap case**

---

## 🧾 Code

```python
class Solution:
    def isScramble(self, s1: str, s2: str) -> bool:
        if len(s1) != len(s2):
            return False
        if s1 == "":
            return True
        return self.solve(s1, s2)

    def solve(self, s1, s2):
        if s1 == s2:
            return True

        if len(s1) <= 1:
            return False

        # Pruning
        if sorted(s1) != sorted(s2):
            return False

        n = len(s1)

        for i in range(1, n):
            # swap case
            if self.solve(s1[:i], s2[n-i:]) and \
               self.solve(s1[i:], s2[:n-i]):
                return True

            # no swap case
            if self.solve(s1[:i], s2[:i]) and \
               self.solve(s1[i:], s2[i:]):
                return True

        return False
```

---

## ⏱️ Complexity

- Time: Exponential ❌  
- Space: recursion stack  

👉 Will TLE

---

# ⚡ Approach 2: Memoization (Top-Down DP)

## 💡 Idea

Store results for:
```
(s1, s2)
```

---

## 🧾 Code

```python
class Solution:
    def isScramble(self, s1: str, s2: str) -> bool:
        if len(s1) != len(s2):
            return False
        if s1 == "":
            return True

        self.dp = {}
        return self.solve(s1, s2)

    def solve(self, s1, s2):
        if s1 == s2:
            return True

        if len(s1) <= 1:
            return False

        # Pruning
        if sorted(s1) != sorted(s2):
            return False

        key = (s1, s2)

        if key in self.dp:
            return self.dp[key]

        n = len(s1)

        for i in range(1, n):
            # swap case
            if self.solve(s1[:i], s2[n-i:]) and \
               self.solve(s1[i:], s2[:n-i]):
                self.dp[key] = True
                return True

            # no swap case
            if self.solve(s1[:i], s2[:i]) and \
               self.solve(s1[i:], s2[i:]):
                self.dp[key] = True
                return True

        self.dp[key] = False
        return False
```

---

## ⏱️ Complexity

- Time: ~ `O(n^4)` ⚠️  
- Space: `O(n^2)`  

👉 Acceptable due to pruning + memoization

# 🔥 Summary

| Approach | Time | Space | Status |
|--------|------|------|--------|
| Recursion | Exponential ❌ | Stack | TLE |
| Memoization | ~O(n⁴) ⚠️ | O(n²) | Accepted |

---

# 🧠 Core Logic

For every split `i`:

### 1️⃣ No Swap
```
s1[:i] ↔ s2[:i]
s1[i:] ↔ s2[i:]
```

### 2️⃣ Swap
```
s1[:i] ↔ s2[n-i:]
s1[i:] ↔ s2[:n-i]
```

---

# ⚠️ Common Mistakes

- Forgetting swap case ❌  
- Missing pruning (`sorted`) ❌  
- No memoization ❌  
- Wrong substring slicing ❌
