# 🧩 416. Partition Equal Subset Sum

## 🔴 Problem
Given an integer array `nums`, return `true` if you can partition the array into two subsets such that the sum of elements in both subsets is equal.

---

## ✨ Example

Input:
    nums = [1,5,11,5]

Output:
    true

Explanation:
    The array can be partitioned as:
    [1, 5, 5] and [11]

---

# 🧠 Key Idea

👉 Total sum must be even  
👉 Convert problem to:

    "Can we find a subset with sum = total / 2 ?"

---

# 🚀 Approach 1: Recursion (Brute Force)

## 💡 Idea
For each element:
    - Include it
    - Exclude it

## 💻 Code

    class Solution:
        def subsetSum(self, nums, s, n):
            if s == 0:
                return True
            if n == 0:
                return False

            if nums[n - 1] <= s:
                return (self.subsetSum(nums, s - nums[n - 1], n - 1) or
                        self.subsetSum(nums, s, n - 1))
            else:
                return self.subsetSum(nums, s, n - 1)

        def canPartition(self, nums):
            total = sum(nums)
            if total % 2 != 0:
                return False
            return self.subsetSum(nums, total // 2, len(nums))

## ⏱️ Complexity
    Time: O(2^n)
    Space: O(n)

---

# 🚀 Approach 2: Memoization (Top-Down DP)

## 💡 Idea
Store subproblem results to avoid recomputation

## 💻 Code

    class Solution:
        def subsetSum(self, nums, s, n, mem):
            if s == 0:
                return True
            if n == 0:
                return False

            if mem[n][s] != -1:
                return mem[n][s]

            if nums[n - 1] <= s:
                mem[n][s] = (
                    self.subsetSum(nums, s - nums[n - 1], n - 1, mem) or
                    self.subsetSum(nums, s, n - 1, mem)
                )
            else:
                mem[n][s] = self.subsetSum(nums, s, n - 1, mem)

            return mem[n][s]

        def canPartition(self, nums):
            total = sum(nums)
            if total % 2 != 0:
                return False

            n = len(nums)
            mem = [[-1 for _ in range(total + 1)] for _ in range(n + 1)]

            return self.subsetSum(nums, total // 2, n, mem)

## ⏱️ Complexity
    Time: O(n * target)
    Space: O(n * target)

---

# 🚀 Approach 3: Bottom-Up DP (2D)

## 💡 Idea

    t[i][j] = can we form sum j using first i elements

## 💻 Code

    class Solution:
        def canPartition(self, nums):
            total = sum(nums)
            if total % 2 != 0:
                return False

            target = total // 2
            n = len(nums)

            t = [[False] * (target + 1) for _ in range(n + 1)]

            for i in range(n + 1):
                t[i][0] = True

            for i in range(1, n + 1):
                for j in range(1, target + 1):
                    if nums[i - 1] <= j:
                        t[i][j] = t[i - 1][j - nums[i - 1]] or t[i - 1][j]
                    else:
                        t[i][j] = t[i - 1][j]

            return t[n][target]

## ⏱️ Complexity
    Time: O(n * target)
    Space: O(n * target)

---

# 🚀 Approach 4: Space Optimized DP (1D) 🔥

## 💡 Idea

    dp[j] = can we form sum j so far

👉 Update in reverse to avoid reusing same element

## 💻 Code

    class Solution:
        def canPartition(self, nums):
            total = sum(nums)
            if total % 2 != 0:
                return False

            target = total // 2
            dp = [False] * (target + 1)
            dp[0] = True

            for num in nums:
                for j in range(target, num - 1, -1):
                    dp[j] = dp[j] or dp[j - num]

            return dp[target]

## ⏱️ Complexity
    Time: O(n * target)
    Space: O(target)

---

# 🎯 Final Summary

- Recursion → try all subsets (slow)
- Memoization → avoids recomputation
- 2D DP → structured solution
- 1D DP → optimized and preferred

---

# 🧠 Pattern

👉 This is a classic:
    0/1 Knapsack (Subset Sum variation)

---

# 🔥 Key Insight

👉 Each element:
    either contributes to the sum OR not

👉 Goal:
    build target sum step by step
