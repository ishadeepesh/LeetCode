# 🎯 494. Target Sum

## 🧩 Problem Statement

You are given an integer array `nums` and an integer `target`.

You want to build an expression by adding either `'+'` or `'-'` before each number and then concatenating them.

### ✨ Example:
nums = [2,1]  
Possible expressions:  
+2+1 = 3  
+2-1 = 1  
-2+1 = -1  
-2-1 = -3  

Return the **number of ways** to reach the target.

---

## 📌 Example

### Input:
nums = [1,1,1,1,1]  
target = 3  

### Output:
5  

### 💡 Explanation:
-1 + 1 + 1 + 1 + 1 = 3  
+1 - 1 + 1 + 1 + 1 = 3  
+1 + 1 - 1 + 1 + 1 = 3  
+1 + 1 + 1 - 1 + 1 = 3  
+1 + 1 + 1 + 1 - 1 = 3  

---

## 🧠 Key Idea (Important Insight)

Instead of assigning `+` and `-`, we convert the problem into a **Subset Sum problem**.

Let:
- `P` = sum of positive numbers  
- `N` = sum of negative numbers  

We know:
P - N = target  
P + N = total_sum  

Adding both:
2P = target + total_sum  
P = (target + total_sum) / 2  

👉 So the problem becomes:

> Count subsets with sum = `(target + total_sum) / 2`

---

## ⚠️ Edge Cases

- If `(target + total_sum)` is **odd** → ❌ No solution  
- If required sum is **negative or too large** → ❌ No solution  

---

## 🚀 Approach

1. Convert problem → subset sum  
2. Use **1D DP array**  
3. Count number of ways to reach target sum  

---

## 🧾 Code (Python)

```python
from typing import List

class Solution:
    def perfectSum(self, arr, target):
        dp = [0] * (target + 1)
        dp[0] = 1

        for num in arr:
            for j in range(target, num - 1, -1):
                dp[j] += dp[j - num]

        return dp[target]

    def findTargetSumWays(self, nums: List[int], target: int) -> int:
        total = sum(nums)

        if (target + total) % 2 != 0:
            return 0

        targetSum = (target + total) // 2

        if targetSum < 0 or targetSum > total:
            return 0

        return self.perfectSum(nums, targetSum)
```

---

## ⏱️ Complexity

- **Time Complexity:** `O(n * target)`  
- **Space Complexity:** `O(target)`  

---

## 🔥 Summary

- Convert sign problem → subset sum  
- Use DP to count subsets  
- Watch edge cases carefully
