# 🚕 2008. Maximum Earnings From Taxi

## 🔴 Problem
You are driving a taxi on a road with `n` points labeled from `1` to `n`.

You are given a list of rides, where each ride is represented as:
    [start, end, tip]

- You may pick up a passenger at `start` and drop them at `end`
- You earn:
      (end - start + tip)
- You can only take **non-overlapping rides**

👉 Return the **maximum earnings** you can achieve.

---

## ✨ Example

**Input:**
    rides = [[2,5,4],[1,5,1],[5,6,2]]

**Output:**
    10

**Explanation:**
    Best choice:
    [2,5,4] → profit = 7
    [5,6,2] → profit = 3

    Total = 10

---

## 🚀 Approach: DP + Binary Search (Weighted Interval Scheduling)

### 💡 Key Idea
- This is a **non-overlapping interval + profit maximization** problem
- Use **Dynamic Programming**
- Use **Binary Search** to find the last compatible ride efficiently

---

## ⚙️ Steps

1. Sort rides based on **end time**
2. Create an array `ends[]` for binary search
3. Define:
       dp[i] = maximum profit till i-th ride
4. For each ride:
   - Option 1: Skip → `dp[i-1]`
   - Option 2: Take → `profit + dp[j]`
     where `j` is the last non-overlapping ride
5. Use `bisect_right` to find `j`

---

## 🔍 Binary Search Trick

To find last ride where:
    end <= start

Use:
    j = bisect_right(ends, start) - 1

---

## ⏱️ Complexity

- **Time:** O(n log n)
- **Space:** O(n)

---

## 💻 Code

```python
from bisect import bisect_right

class Solution:
    def maxTaxiEarnings(self, n, rides):
        # Step 1: Sort by end time
        rides.sort(key=lambda x: x[1])

        # Step 2: Prepare arrays
        ends = [ride[1] for ride in rides]
        dp = [0] * len(rides)

        for i in range(len(rides)):
            start, end, tip = rides[i]
            profit = end - start + tip

            # Step 3: Find last non-overlapping ride
            j = bisect_right(ends, start) - 1

            take = profit
            if j >= 0:
                take += dp[j]

            skip = dp[i - 1] if i > 0 else 0

            dp[i] = max(take, skip)

        return dp[-1]
