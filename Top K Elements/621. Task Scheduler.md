# 🔎 621. Task Scheduler

## 🧩 Problem

You are given an array `tasks` where each task is represented by a capital letter (`A-Z`), and an integer `n`.

🎯 **Goal:**
Find the **minimum number of CPU intervals** needed to finish all tasks such that:

> 🔑 Same tasks must have at least **n intervals gap**

Each interval can be:

* A task execution
* OR an idle slot

---

## 📌 Example

**Input**

```python id="c7z6l1"
tasks = ["A","A","A","B","B","B"]
n = 2
```

**Output**

```text id="2x6d1f"
8
```

**Explanation**

```text id="yzc8x0"
A → B → idle → A → B → idle → A → B
```

---

# 💡 Key Insight (MOST IMPORTANT)

👉 The most frequent task determines the structure.

Let:

* `maxFreq` = highest frequency
* `maxCount` = number of tasks having that frequency

---

# 🧠 Optimal Formula

```text id="m4m7xg"
Answer = max(
    total_tasks,
    (maxFreq - 1) * (n + 1) + maxCount
)
```

---

# 🔍 Why this works

### Imagine arranging the most frequent task:

```text id="3gq2o1"
A _ _   A _ _   A
```

* `(maxFreq - 1)` blocks
* Each block has size `(n + 1)`
* Last block only needs `maxCount` slots

---

# 💻 Python Implementation (Optimal)

```python id="3n9d4v"
from collections import Counter

class Solution:
    def leastInterval(self, tasks, n):
        count = Counter(tasks)

        maxFreq = max(count.values())
        maxCount = list(count.values()).count(maxFreq)

        return max(len(tasks), (maxFreq - 1) * (n + 1) + maxCount)
```

### Step 2: Apply formula

```python id="g2h1pd"
(maxFreq - 1) * (n + 1) + maxCount
= (3 - 1) * (2 + 1) + 2
= 2 * 3 + 2
= 8
```

# ⚖️ Why this is optimal

* No heap needed
* No simulation required
* Direct mathematical computation

---

# ⚠️ Edge Cases

✔ `n = 0` → no restriction → return `len(tasks)`
✔ All tasks same → lots of idle slots
✔ Multiple max frequency tasks
