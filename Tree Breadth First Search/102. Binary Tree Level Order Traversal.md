# üå≥ 102. Binary Tree Level Order Traversal

## üìù Problem Statement

Given the `root` of a binary tree, return the **level order traversal** of its nodes' values ‚Äî from **left to right**, **level by level**.

---

## üì• Example

**Input:**
root = [3,9,20,null,null,15,7]

**Output:**
[[3],[9,20],[15,7]]

**Explanation:**
#### Level 0: 3

#### Level 1: 9 20

#### Level 2: 15 7

#### Level order traversal:
‚Üí [[3], [9,20], [15,7]]

---

## üí° Approach: BFS with Queue

1. Use a **queue** to perform **Breadth-First Search (BFS)**.
2. Start with the root node in the queue.
3. For each level:
   - Process all nodes currently in the queue.
   - Append their children (left and right) to the queue.
   - Add their values to the result for the current level.

---

## ‚úÖ Python Code

```python
class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []

        q = [root]
        res = []

        while q:
            levelSize = len(q)
            level = []

            for _ in range(levelSize):
                node = q.pop(0)
                level.append(node.val)
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)

            res.append(level)

        return res
```
‚è±Ô∏è Time Complexity
O(n): Each node is visited once, where n is the number of nodes in the tree.

üíæ Space Complexity
O(n): The queue can hold up to n nodes in the worst case (e.g., a complete binary tree).
